vue踩坑之downloading templates

发表于 2018-11-27 | 分类于 开发

vue 初始化时遇到这种情况:Failed to download repo vuejs-templates/webpack: read ECONNRESET

原因是templates一直下载不下来，试了好多种方法，更新node版本，重新install等等都不能成功。后来试了一下离线安装templates,可以成功安装初始化项目。
首先克隆vuejs-templates到本地：git clone https://github.com/vuejs-templates/simple ~/.vue-templates/simple
离线初始化项目：vue init simple my-project –offline
